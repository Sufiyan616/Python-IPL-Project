# IPL 2022 Data Analysis Project (Python EDA)

## Project Overview
This project analyzes IPL 2022 match data using Python.
The goal is to understand team performance, match outcomes, and key patterns
from the tournament using simple data analysis techniques.

---

## Objective
- To analyze IPL 2022 match data
- To find which teams performed better
- To understand the impact of toss, scores, and match results
- To practice real-world data analysis using Python

---

## Dataset Information
- Total Matches: 74
- Total Columns: 20
- Data Type: Match-level IPL 2022 data
- No missing values in the dataset

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Analysis Performed
- Data loading and inspection
- Checking rows, columns, and data types
- Missing value analysis
- Team-wise match wins
- Match result analysis
- Toss decision analysis
- Score-based comparisons
- Visualization using bar charts and plots

---

## Key Insights
- Some teams won more matches consistently in IPL 2022
- Teams choosing to field after winning the toss often performed better
- High first-innings scores increased winning chances
- Match outcomes varied based on venue and match conditions
